# Returns Policy (Extended)

## Purpose

This policy defines how and when customers can return purchased items. The assistant must follow this document strictly and apply tier and mode rules when offering options.

---

## Standard Return Window

* Returns are accepted within **14 calendar days** from the **purchase date**.
* The return window starts on the day of purchase, not delivery (demo rule).
* Requests submitted after the window are normally rejected unless goodwill rules apply.

---

## Item Condition Requirements

To be eligible for return, the item must:

* Be unused and unworn
* Have original tags attached (if originally provided)
* Be free of stains, odors, or damage
* Include original packaging when required (e.g., accessories, premium items)

---

## Non-Returnable Items

The following items are non-returnable unless defective or wrongly shipped:

* Intimate apparel / underwear
* Personal care or hygiene products
* Customized or personalized items
* Final sale / clearance items (if marked as such)

---

## Eligible Return Reasons

* **Changed Mind**
* **Wrong Size / Fit Issue**
* **Wrong Item Received**
* **Item Defective or Damaged**

Each reason triggers different handling rules.

---

## Return Shipping Rules

### Defective or Wrong Item

* Free return shipping for all tiers
* Customer must provide photo evidence if requested
* Priority handling for Gold and VIP

### Changed Mind / Size Issue

* Bronze: customer pays return shipping
* Silver: customer pays return shipping
* Gold: free return shipping
* VIP: free return shipping

---

## Return Request Process

1. Customer provides:

   * Customer ID
   * Item name
   * Purchase date
   * Reason for return
2. Assistant checks eligibility
3. Return instructions are issued
4. Customer ships item
5. Warehouse inspects item
6. Outcome is determined (refund, exchange, store credit, or rejection)

---

## Inspection Outcomes

### Approved

* Item meets all conditions
* Process continues to refund/exchange/store credit

### Partial Approval

* Minor wear, missing tags, or packaging
* Partial refund or store credit may be issued

### Rejected

* Item used, damaged by customer, or outside goodwill limits
* Item is returned to customer or disposed according to policy

---

## Late Returns

* Returns requested up to **7 days late** may be considered under goodwill rules.
* Beyond 7 days late: return is rejected.

---

## Fraud and Abuse

If repeated suspicious behavior is detected:

* Returns may be denied
* Loyalty benefits may be suspended
* Account may be flagged for review
